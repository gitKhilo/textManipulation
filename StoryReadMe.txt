What this code does is basic text file manipulaton on python - takes Story.txt and:

a) It reads every odd line of the text file and displays it on the screen.  
b) It reads every even line of the text file and displays it on the screen.  
c) It appends the phrase “Once upon a time,” and a blank line under it, to the beginning of the text file.  
d) It appends a new line and then the word “the End” to the end of the text file.  
e) It copies the first 4 lines and writes them to another text file that it creates programmatically.  
f) It deletes the first line of the new file.
